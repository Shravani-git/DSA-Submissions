# [Median of an Array](https://www.geeksforgeeks.org/problems/find-the-median0527/1?page=2&difficulty=Basic&sortBy=submissions)
## Easy
Given an array arr[] of integers, calculate the median.
Examples:
Input: arr[] = [90, 100, 78, 89, 67]
Output: 89
Explanation: After sorting the array middle element is the median 

Input: arr[] = [56, 67, 30, 79]
Output: 61.5
Explanation: In case of even number of elements, average of two middle elements is the median. 
Input:&nbsp;arr[] = [1, 2]Output: 1.5Explanation: The average of both elements will result in 1.5.
Constraints:1 &lt;= arr.size() &lt;= 1051 &lt;= arr[i] &lt;= 105
